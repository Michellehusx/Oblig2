Git og Github

Både Git og Github er en plattform hvor en eller flere kan jobbe med prosjekter innenfor programmering , designing osv.Git er mer brukt for å programmerere på et program.Mens Github er et nettsted som gjør mye av den samme jobben men siden platformen er på et nettsted er det letter å komme i kontakt med andre.Begge platformene gjør at du og ditt team kan gjøre endringer på koden , programvaren eller nettsiden dere utvikler fra hvor som helst i verden.
